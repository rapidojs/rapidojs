🚀 Rapido.js 多模块 API 测试

👤 用户模块 (UserModule)
GET /users
获取所有用户列表
GET /users/:id
根据 ID 获取用户详情
POST /users
创建新用户（需要 name 和 email）
PUT /users/:id
更新用户信息
DELETE /users/:id
删除用户
📦 产品模块 (ProductModule)
GET /products
获取所有产品（支持 category 和 inStock 查询参数）
GET /products/in-stock
获取所有有库存的产品
GET /products/:id
根据 ID 获取产品详情
POST /products
创建新产品（需要 name, description, price, category, inStock）
PUT /products/:id
更新产品信息
DELETE /products/:id
删除产品
🔐 认证模块 (AuthModule)
GET /auth/demo-users
获取演示用户账号信息
POST /auth/login
用户登录（需要 email 和 password）
POST /auth/refresh
刷新令牌（需要 Authorization header）
GET /auth/profile
获取用户资料（需要 Authorization header）
